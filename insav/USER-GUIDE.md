# insav 自动刷邀请机器人使用教程

## 机器人简介

本机器人可以自动为您的 insav 账号刷邀请，快速获取会员权限。

会员有效期约 1 小时，建议用完再来，避免浪费。

## 使用前准备

1. 加入群组：@GithubYu9191
   - 必须先加入群组才能使用机器人
   - 这是使用机器人的必要条件

2. 准备好您的账号密码
   - 确保账号已在对应平台注册
   - 记住您的用户名和密码

## 支持的平台

- 主站
- 无码站
- 动漫站
- 4K站

注意：只有无码站支持自定义域名更新

## 使用步骤

### 第一步：启动机器人

发送命令：/start

机器人会显示欢迎信息和平台选择按钮

### 第二步：选择平台

点击对应的平台按钮：
- [主站]
- [无码站]
- [动漫站]
- [4K站]

### 第三步：发送账号密码

按照以下格式发送您的账号密码：

```
账号 密码
```

示例：
```
myusername mypassword
```

注意事项：
- 账号和密码之间用空格分隔
- 不要包含其他字符
- 确保账号已在对应平台注册

### 第四步：等待处理

机器人会自动执行以下操作：
1. 登录您的账号
2. 获取邀请码
3. 注册小号进行邀请
4. 检查会员状态
5. 返回处理结果

整个过程大约需要 10-20 秒

## 处理结果说明

### 成功情况

机器人会显示：
- 平台名称
- 您的账号
- 注册统计（成功/失败数量）
- 会员状态（已开通/未开通）
- 到期时间
- 详细日志

### 已有会员

如果检测到您已经有会员，机器人会提示：
```
检测到您已有会员
到期时间: XXXX年XX月XX日 XX:XX:XX
请不要贪心哦，到期再来！
```

此时不会进行刷邀请操作，请等到会员到期后再使用。

### 失败情况

可能的失败原因：
1. 账号密码错误
2. 账号未在该平台注册
3. 网络问题
4. 域名失效（无码站）

错误提示：
- "请检查选择的站点是否已注册账号，本次登录验证失败"
  - 说明账号可能未在该平台注册
  - 或者选择的平台不正确

## 更新无码站域名

由于无码站域名可能会变化，机器人支持自定义域名。

### 操作步骤

1. 点击 [更新无码站域名] 按钮

2. 发送新的域名，支持以下格式：
   ```
   https://wuma.qmag372.com/
   https://wuma.qmag372.com
   wuma.qmag372.com
   ```

3. 机器人确认域名更新成功后，即可使用新域名

4. 然后正常选择无码站并发送账号密码

## 常见问题

### Q1: 提示未加入频道怎么办？

A: 请先加入频道 @GithubYu9191，加入后重新发送 /start

### Q2: 会员有效期是多久？

A: 约 1 小时，建议用完再来刷新

### Q3: 可以频繁刷邀请吗？

A: 不建议频繁刷邀请，建议等会员到期后再使用

### Q4: 为什么提示"请不要贪心"？

A: 因为检测到您的账号已经有会员且未到期，请等到期后再来

### Q5: 登录失败怎么办？

A: 请检查：
- 账号密码是否正确
- 是否在对应平台注册过账号
- 选择的平台是否正确
- 无码站域名是否需要更新

### Q6: 可以在群组使用吗？

A: 不可以，机器人只响应私聊消息

### Q7: 如何取消当前操作？

A: 发送命令 /cancel 即可取消

### Q8: 刷邀请会注册几个小号？

A: 每次会注册 2 个小号进行邀请

### Q9: 所有平台都能刷吗？

A: 是的，支持主站、无码站、动漫站、4K站

### Q10: 域名更新后会保存吗？

A: 域名更新是临时的，每次使用时如果域名变化需要重新更新

## 注意事项

1. 会员有效期短
   - 约 1 小时有效期
   - 建议用完再来
   - 不要浪费

2. 不要频繁使用
   - 避免频繁刷邀请

   - 账号必须在对应平台注册
   - 无码站域名可能需要更新

5. 群组限制
   - 机器人只在私聊中工作
   - 群组消息会被忽略
   - 请私聊机器人使用
