<details><summary><b>功能</b></summary>

- VIP 视频解锁（长视频 / 短视频 / 次数耗尽均可播放）
- 全量去广告（启动广告 / 播放前广告 / 页面广告 / 弹窗 / 轮播）
- 伪装 VIP 会员（昵称 / 金币 / 剩余天数）
- 屏蔽棋牌 / 游戏入口
- 外部播放器跳转（可选，支持 13 款播放器 + 自定义 scheme）

</details>
